# 项目描述   
实现用mpv播放B站视频，支持弹幕。   

# 功能   
1. 获取视频和音频的真实地址。  
2. 能在浏览器显示调用选项。  
3. 支持弹幕显示。

# 实现方法  
## 获取视频和音频的真实地址  
通过 you-get 获取，将其输出结果保存到本地data.txt文件中。

## 在浏览器显示调用MPV选项  
脚本实现。

## 支持弹幕
通过将 you-get 获取的弹幕文件转换成 ass 字幕文件，然后加载进 mpv。
